# 天気AI感想アプリ 開発ハンズオン解説書

React + MUI + OpenAI + Open-Meteo API を使って「今日の天気にAIが感想を言う」アプリを作る。

---

## 完成イメージ

- 画面を開くと現在地の天気を自動取得
- 取得した天気データをOpenAIに渡し、AIが一言コメントを生成
- MUIで見やすくカード表示

---

## 技術スタック

| 役割 | 採用技術 | 理由 |
|------|---------|------|
| フレームワーク | React (Vite) | 高速DX |
| UI | Material UI (MUI) v6 | リッチなコンポーネント群 |
| 天気API | Open-Meteo | 完全無料・APIキー不要 |
| AI | OpenAI API (`gpt-4o-mini`) | 安価で十分な品質 |
| HTTP | fetch (標準) | 依存最小化 |

## 5. 天気取得フック

`src/hooks/useWeather.js`

```js
import { useState, useEffect } from "react";

export function useWeather() {
  const [weather, setWeather] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    navigator.geolocation.getCurrentPosition(
      async ({ coords }) => {
        try {
          const { latitude, longitude } = coords;
          const url =
            `https://api.open-meteo.com/v1/forecast` +
            `?latitude=${latitude}&longitude=${longitude}` +
            `&current=temperature_2m,apparent_temperature,weathercode,windspeed_10m,relativehumidity_2m` +
            `&timezone=Asia%2FTokyo`;

          const res = await fetch(url);
          if (!res.ok) throw new Error("天気データの取得に失敗しました");
          const data = await res.json();

          setWeather({
            lat: latitude,
            lon: longitude,
            temp: data.current.temperature_2m,
            feelsLike: data.current.apparent_temperature,
            code: data.current.weathercode,
            wind: data.current.windspeed_10m,
            humidity: data.current.relativehumidity_2m,
          });
        } catch (e) {
          setError(e.message);
        } finally {
          setLoading(false);
        }
      },
      (err) => {
        setError("位置情報の取得を許可してください: " + err.message);
        setLoading(false);
      }
    );
  }, []);

  return { weather, loading, error };
}
```

## 6. OpenAI呼び出しフック

`src/hooks/useAiComment.js`

```js
import { useState } from "react";
import OpenAI from "openai";
import { weatherCodeMap } from "../utils/weatherCodeMap";

const client = new OpenAI({
  apiKey: import.meta.env.VITE_OPENAI_API_KEY,
  dangerouslyAllowBrowser: true, // 開発用。本番はバックエンド経由推奨
});

export function useAiComment() {
  const [comment, setComment] = useState("");
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState(null);

  async function generateComment(weather) {
    setLoading(true);
    setError(null);

    const { label } = weatherCodeMap[weather.code] ?? { label: "不明" };
    const prompt =
      `今日の天気は「${label}」、気温${weather.temp}℃（体感${weather.feelsLike}℃）、` +
      `湿度${weather.humidity}%、風速${weather.wind}km/hです。` +
      `この天気について、ユーモアを交えた一言感想を日本語で50〜80文字で答えてください。`;

    try {
      const response = await client.chat.completions.create({
        model: "gpt-4o-mini",
        messages: [
          {
            role: "system",
            content: "あなたは天気に詳しいユーモアのある気象コメンテーターです。",
          },
          { role: "user", content: prompt },
        ],
        max_tokens: 150,
      });

      setComment(response.choices[0].message.content.trim());
    } catch (e) {
      setError("AIコメントの生成に失敗しました: " + e.message);
    } finally {
      setLoading(false);
    }
  }

  return { comment, loading, error, generateComment };
}
```

> **本番運用の注意**: `dangerouslyAllowBrowser: true` はAPIキーがブラウザに露出します。
> 本番ではExpressなどのバックエンドAPIを経由してOpenAIを呼ぶ構成にしてください。

## 8. AiCommentコンポーネント

`src/components/AiComment.jsx`

```jsx
import {
  Card, CardContent, Typography, Box,
  Button, CircularProgress, Alert,
} from "@mui/material";
import AutoAwesomeIcon from "@mui/icons-material/AutoAwesome";

export function AiComment({ comment, loading, error, onGenerate }) {
  return (
    <Card elevation={3} sx={{ borderRadius: 3, mt: 2 }}>
      <CardContent>
        <Box display="flex" alignItems="center" gap={1} mb={2}>
          <AutoAwesomeIcon color="primary" />
          <Typography variant="h6" fontWeight="bold">AIの一言</Typography>
        </Box>

        {error && <Alert severity="error" sx={{ mb: 2 }}>{error}</Alert>}

        {comment ? (
          <Typography variant="body1" sx={{ lineHeight: 1.8, mb: 2 }}>
            {comment}
          </Typography>
        ) : (
          <Typography variant="body2" color="text.secondary" sx={{ mb: 2 }}>
            ボタンを押すとAIが今日の天気に感想を言います
          </Typography>
        )}

        <Button
          variant="contained"
          onClick={onGenerate}
          disabled={loading}
          startIcon={loading ? <CircularProgress size={16} /> : <AutoAwesomeIcon />}
          fullWidth
        >
          {loading ? "生成中..." : comment ? "もう一度生成" : "感想を生成"}
        </Button>
      </CardContent>
    </Card>
  );
}
```

## 9. App.jsx（メイン画面）

`src/App.jsx`

```jsx
import {
  Container, Typography, Box,
  CircularProgress, Alert, CssBaseline,
} from "@mui/material";
import { ThemeProvider, createTheme } from "@mui/material/styles";
import { useWeather } from "./hooks/useWeather";
import { useAiComment } from "./hooks/useAiComment";
import { WeatherCard } from "./components/WeatherCard";
import { AiComment } from "./components/AiComment";

const theme = createTheme({
  palette: {
    primary: { main: "#1976d2" },
    background: { default: "#f0f4f8" },
  },
  typography: {
    fontFamily: "'Noto Sans JP', sans-serif",
  },
});

export default function App() {
  const { weather, loading: weatherLoading, error: weatherError } = useWeather();
  const { comment, loading: aiLoading, error: aiError, generateComment } = useAiComment();

  return (
    <ThemeProvider theme={theme}>
      <CssBaseline />
      <Container maxWidth="sm" sx={{ py: 4 }}>
        <Typography variant="h4" fontWeight="bold" textAlign="center" mb={3}>
          🌤️ 今日の天気
        </Typography>

        {weatherLoading && (
          <Box textAlign="center" py={4}>
            <CircularProgress />
            <Typography mt={2}>位置情報と天気を取得中...</Typography>
          </Box>
        )}

        {weatherError && <Alert severity="error">{weatherError}</Alert>}

        {weather && (
          <>
            <WeatherCard weather={weather} />
            <AiComment
              comment={comment}
              loading={aiLoading}
              error={aiError}
              onGenerate={() => generateComment(weather)}
            />
          </>
        )}
      </Container>
    </ThemeProvider>
  );
}
```

## よくあるエラーと対処

| エラー | 原因 | 対処 |
|--------|------|------|
| 位置情報が取得できない | ブラウザの許可設定 | アドレスバーの鍵アイコンから許可 |
| 401 Unauthorized | OpenAI APIキー誤り | `.env` のキーを確認 |
| CORS エラー | Open-MeteoはCORSフリーなので基本発生しない | — |
| `dangerouslyAllowBrowser` 警告 | 本番非推奨の設定 | バックエンドAPI化で解決 |
